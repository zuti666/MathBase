# 离散型随机变量及其分布律

有些随机变量,它全部可能取到的值是有限个或可列无限多个,这种随机变量称为**离散型随机变量**。要掌握一个离散型随机变量X的统计规律,**必须且只需知道X的所有可能取值以及取每一个可能值的概率**。
